# UI Icons And Visual QA

- Use `lucide-react` SVG icons for controls, navigation, and product UI by default.
- Emoji are content/tone choices, not default app chrome.
- Verify UI changes on mobile and desktop viewports.
- Keep text inside controls readable and non-overlapping.
- Do not add viewport-scaled typography.
- Use standard Tailwind/shadcn font-size utilities for normal product UI.
